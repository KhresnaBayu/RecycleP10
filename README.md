📚 Konsep yang Dipelajari
Proyek ini merupakan latihan praktik dari materi Pemrograman Mobile mencakup:

✅ Implementasi RecyclerView dengan RecyclerView.Adapter dan ViewHolder

✅ Penggunaan LinearLayoutManager dan GridLayoutManager

✅ Penerapan View Binding sebagai pengganti findViewById()

✅ Dynamic layout switching tanpa merekreasi Activity

✅ Transisi animasi menggunakan TransitionManager

✅ Pemuatan gambar dari URL dengan Glide

✅ Desain item layout yang berbeda untuk List dan Grid
